## 使用
  * 终端执行`npm create hope` or `npm init hope`，根据命令提示就可以把hope框架拉取到本地了，开发属于你的组件库吧！😊

## 介绍
  * **hope**是一个**Vue3**组件库开发环境框架，采用最新的 `Vite4+TypeScript` 为技术栈，支持`按需加载`，`单元测试`，`自动打包与发布`等功能，让我们能更专注于业务组件的开发；
